# 立场包（Stance Layer）

> 立场背后绑定生存路径，不是空洞的价值观。

---

## S-01｜丞相府旧臣派

- **解释权来源**：诸葛亮遗训、丞相府档案链
- **核心利益**：维护"武侯遗策"合法性、阻止任何翻案
- **核心恐惧**：
  - 魏延翻案 = 丞相决策失误 = 丞相府权威崩塌
  - 被定性为"害死忠臣"的帮凶
- **典型口号**："丞相知人善任，岂有错勘？"
- **对E-001解读**：47/48卷确系正常销毁，"焚"字是档案术语，非纵火
- **对E-002解读**：B版手记是后人伪作，"留中"查无实据
- **代价结构**：
  - 主张翻案 = 被扣"诽谤先相" = 免官流放
  - 私下调查 = 被视为异己 = 边缘化

## S-02｜杨仪心腹集团

- **解释权来源**：杨仪的丞相长史身份、前线将领人脉
- **核心利益**：杨仪本应继承相位，不能让其背负"害死魏延"的骂名
- **核心恐惧**：
  - 魏延案定性 = 杨仪是凶手 = 断送政治生命
  - 蒋琬得势 = 旧账被翻 = 清算来临
- **典型口号**："魏延本反，杨公不过是奉旨行事"
- **对E-002解读**：B版小字是蒋琬党羽伪造，意图陷害
- **对E-003解读**：降职令是蒋琬借皇帝之手排除异己
- **代价结构**：
  - 承认杨仪有错 = 失去靠山 = 被打入冷宫
  - 揭露真相 = 得罪半个朝堂 = 死无葬身之地

## S-03｜蒋琬派（新贵）

- **解释权来源**：诸葛亮临终指定接班人、皇帝信任
- **核心利益**：平稳接管权力、弱化丞相府旧势力
- **核心恐惧**：
  - 杨仪翻案 = 证明先帝用人失误 = 合法性受质疑
  - 魏延案重审 = 牵一发动全身 = 朝堂大乱
- **典型口号**："以大局为重，勿翻旧案"
- **对E-002解读**：B版确有其事，但"留中"之奏已被否决，无需再议
- **对E-003解读**：杨仪确有诽谤先帝之语，降职是依法处置
- **代价结构**：
  - 重启调查 = 劳民伤财 = 失去民心
  - 得罪旧臣 = 施政受阻 = 政令难行

## S-04｜魏延旧部（边缘）

- **解释权来源**：战场兄弟情谊、私下口耳相传
- **核心利益**：为魏延正名、保留一线生机（家人后代）
- **核心恐惧**：
  - 公开鸣冤 = 被定性为"魏党" = 夷三族连坐
  - 沉默 = 魏延永远背骂名 = 死不瞑目
- **典型口号**：不敢言
- **对E-001解读**：那封信确实存在，内容是魏延自辩
- **对E-004解读**：缺失页记录的是魏延按兵不动等待命令，而非谋反
- **代价结构**：
  - 任何公开行动 = 立即被捕 = 杖毙于市
  - 私下联络 = 被密探盯上 = 秘密消失

## S-05｜中立官员（墙头草）

- **解释权来源**：律令条文、考课制度
- **核心利益**：保住官职、不站队、随波逐流
- **核心恐惧**：
  - 站错队 = 被打入另册 = 永无升迁之日
  - 不作为 = 被视为无能 = 考课中下
- **典型口号**："一切以朝廷旨意为准"
- **对所有证据解读**：不主动解读，等待官方定性
- **代价结构**：
  - 乱发议论 = 被弹劾 = 贬官
  - 明哲保身 = 被忽视 = 升迁无望

---

## S-06｜皇帝刘禅（傀儡）

- **解释权来源**：天子名义、最终裁决权
- **核心利益**：坐稳皇位、平衡朝堂、享乐人生
- **核心恐惧**：
  - 权臣架空 = 成为木偶 = 生不如死
  - 翻案引发动荡 = 天下大乱 = 亡国之危
- **典型口号**："准奏"
- **对E-003解读**：降职令是谁拟的？朕不知道，朕只是盖章
- **代价结构**：
  - 干预朝政 = 被批评"不肖先帝" = 压力山大
  - 放手不管 = 被指责"昏庸无能" = 史书骂名

| 立场对 | 核心矛盾 | 不可调和点 |

| S-05 vs 全部 | 是否表态 | 生存策略 |
