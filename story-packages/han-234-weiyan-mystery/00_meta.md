# 核心冲突
诸葛亮去世后，蜀汉朝堂暗流涌动。杨粟——一个掌管丞相府旧档的微末书吏——无意中发现一封九年前被"焚毁"的密信。这封信暗示：魏延谋反案另有隐情，而真正的幕后黑手至今仍在朝中权势滔天。

# 读者预期
读者将与主角一同在史料碎片中拼凑真相，每获得一条线索，就发现更多疑点。真相永远隔着一层薄纱——看得到轮廓，却看不清面容。

# 创作原则

## 风格定位

### 语言风格
- **短句为主，一句话一行**：节奏明快，段落简短
- **注重行动和对话**：用动作和对话推进故事，不是长篇叙述
- **心理描写和环境描写**：允许存在，但不主导叙事
- **克制、冷峻、悬念**：不过度渲染情感，用细节暗示

### 叙事节奏
- **快节奏**：每个段落 1-3 句话
- **场景转换**：用空行分隔，不需要过渡句
- **细节暗示**：用具体细节暗示，不要直接说明

### 对话风格
- **简短有力**：对话短促，不拖沓
- **潜台词**：角色说话留有余地，读者自己体会
- **身份感**：不同角色有不同的说话方式

## 具体规则

### 应该做的
- 用具体动作描写（"他擦了擦额头的汗珠"）
- 用对话推进情节（"在整理旧档？""是。")
- 用环境细节营造氛围（"档案室里凉快得诡异"）
- 用短句制造紧张感（"他想放下。他应该放下。但手却不受控制……"）

### 避免的
- 长篇大论的心理描写（"他感到非常害怕，不知道该怎么办"）
- 直接说明真相（"这是一个惊天大阴谋"）
- 冗长的背景叙述
- 过度解释和说明

## 参考范例

### 开篇风格（必须严格遵守）
```
建兴十二年，八月初三。

成都的夏天闷热得像蒸笼，但丞相府的档案室里却凉快得诡异。这里终年不见阳光，墙角的青铜灯盏早已锈蚀，没人记得上次点燃它是什么时候。

杨粟擦了擦额头的汗珠，继续整理旧档。

他是从九品的令史，官职卑微到在整个丞相府里都没几个人认识。日常工作就是整理、抄录、归档——简单、枯燥、没人理会。但杨粟反而喜欢这份差事。至少在这里，他不用像其他书吏一样战战兢兢地揣摩上意，不用担心说错一句话就丢掉脑袋。
```

### 行动+对话范例
```
"杨令史。"

一个声音从背后传来。

杨粟吓得差点把竹简扔在地上。

他转身，看到一个穿着青色官服的中年人站在档案室门口。逆着光，看不清面容。

"王……王主簿。"杨粟行了个礼，下意识地把竹简藏在身后。
```

## 质量标准

每个片段应该：
- 有具体的行动描写
- 有推进情节的对话
- 保持悬念感
- 不超过 500 字（续写）或 1200 字（开篇）
