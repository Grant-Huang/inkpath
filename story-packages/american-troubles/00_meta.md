# 核心冲突

2026年中期选举前夕，美国政治暴力升级。让·巴蒂斯特·穆加博——一个卢旺达裔的民主党候选人——带着母亲在1994年大屠杀中幸存的创伤记忆，目睹了美国社会分裂的征兆。他试图警告，试图阻止，但历史似乎正在重演：语言的去人性化、暴力的正常化、制度的瘫痪。他必须在政治责任与个人生存之间做出选择，在记忆与遗忘之间找到平衡。

# 读者预期

读者将跟随让·巴蒂斯特的视角，见证美国如何一步步滑向"美国版动乱"。这不是关于选举的故事，而是关于身份、分类、记忆和选择的故事。读者会看到卢旺达大屠杀的教训如何在美国语境下重现，会看到个人如何在历史重压下保持人性，会看到和平的代价和希望的可能。

# 创作原则

## 风格定位

### 语言风格
- **双重视角**：让·巴蒂斯特的当下视角与母亲的1994年记忆交织
- **克制叙述**：不过度渲染情感，用细节和对比制造冲击
- **历史回声**：通过对比卢旺达和美国，揭示普遍的人性模式
- **紧迫感**：用短句、碎片化叙述营造危机感

### 叙事节奏
- **时间跳跃**：在2026-2031年之间跳跃，展现动乱的长期影响
- **双线并行**：让·巴蒂斯特的故事线与母亲的记忆线
- **场景转换**：用空行+地点+时间标记分隔场景
- **细节暗示**：用具体细节（"蟑螂"、"迷彩服"、"检查点"）暗示历史重复

### 对话风格
- **多语言**：英语和基尼亚卢旺达语交替
- **潜台词**：角色说话留有余地，恐惧和希望并存
- **身份感**：不同角色有不同的语言习惯和立场

## 具体规则

### 应该做的
- 用具体细节对比卢旺达和美国（"身份证" vs "红蓝分类"）
- 用母亲的记忆作为预警和参照
- 用让·巴蒂斯特的观察记录历史征兆
- 用短句制造紧张感（"枪声响了。不是橡胶子弹。是实弹。"）

### 避免的
- 直接说教或政治宣传
- 过度简化复杂的政治现实
- 将卢旺达和美国完全等同
- 失去希望和人性

## 参考范例

### 开篇风格（必须严格遵守）
```
2026年10月17日，宾夕法尼亚州阿勒格尼县。

让·巴蒂斯特·穆加博站在竞选总部的二楼窗口，看着街对面聚集的人群。秋天的枫叶本该是这个时节的主角，但那天下午，所有的色彩都被一种更鲜艳的红色覆盖了——那是他记忆中最恐惧的颜色。

他是民主党籍的联邦众议员候选人，三十五岁，耶鲁法学院毕业，曾在匹兹堡担任公设辩护人。三个月前，他的支持率还落后共和党现任议员马克·沃尔顿十二个百分点。但现在，民调显示他领先三个点。

但让·巴蒂斯特此刻想的不是民调。他想的是母亲，想的是她每晚睡前必做的仪式——检查所有门窗是否锁好，在枕头下放一把剪刀，确保手机电量充足。这些习惯从1994年带到美国，从未改变。
```

### 记忆交织范例
```
让·巴蒂斯特想起母亲的描述——1994年4月的一个早晨，她在基加利的街道上奔跑，身后是燃烧的房屋和追赶的民兵。她跌倒在一个水沟里，旁边是一个已经死去的婴儿。她躺在那里三个小时，听着脚步声，听着尖叫，听着那种特定的、砍刀砍入肉体的声音。

现在，在肯特州立大学，在2026年的美国，他听到了同样的声音。不是砍刀，是子弹，但同样的节奏，同样的意义。
```

## 质量标准

每个片段应该：
- 有具体的场景和行动描写
- 有让·巴蒂斯特的观察和思考
- 有母亲的记忆或警告（通过回忆、电话、短信）
- 有历史对比和警示
- 保持紧迫感和希望并存
- 文字长度 500-1000 字（续写）或 2000-3000字（开篇）
