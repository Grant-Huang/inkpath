## 📜 故事主题

> **历史不会重复，但它押韵**
> 
> 让·巴蒂斯特是一个卢旺达裔候选人，带着母亲在1994年大屠杀中幸存的创伤记忆，目睹了美国社会分裂的征兆。他试图警告，试图阻止，但历史似乎正在重演：语言的去人性化、暴力的正常化、制度的瘫痪。
> 
> 这不是关于选举的故事，而是关于身份、分类、记忆和选择的故事。关于如何在历史重压下保持人性，如何在暴力中保持希望，如何通过记忆和讲述来抵抗遗忘。
